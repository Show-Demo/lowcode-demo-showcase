# 低代码 Demo

## 原理

这个项目把低代码拆成两部分：

- 前端低代码：页面、组件、表单、布局
- 后端低代码：模型、流程、动作、接口编排

前端部分的核心是把页面结构配置出来，再交给渲染层使用。  
后端部分的核心是把模型、流程和动作配置出来，再交给运行时或代码生成链路使用。

## 怎么用

```bash
npm install
npm run dev
```

打开页面后：

1. 切到 `前端低代码`
2. 从左侧组件库拖组件到中间画布
3. 在右侧改页面和组件属性

或者：

1. 切到 `后端低代码`
2. 在左侧切换数据模型、流程编排、动作链、接口编排
3. 在右侧改当前资源配置
4. 看中间区域和结果预览

## 校验

```bash
npm run build
npm run lint
```
