# 低代码 Showcase

这个仓库放的是一套低代码界面演示。

里面拆了两部分：

- 前端低代码：看页面怎么搭、表单怎么配、组件怎么拖
- 后端低代码：看模型怎么配、流程怎么走、动作和接口怎么编排

重点不是把平台全部做完，而是先把这两类东西的样子做出来：

- 前端低代码大概长什么样
- 后端低代码大概长什么样
- 两边各自负责什么

前端这边能直接看到组件库、页面画布、结构树和属性面板，拖进去的组件可以继续改。后端这边放的是模型设计、流程编排、动作链和接口编排，主要拿来说明后端低代码平时都在配什么。

如果只是想做方案演示、录个概念视频，或者拿它当继续往下扩的平台草图，这个仓库已经够用。
