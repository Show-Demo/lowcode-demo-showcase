# 低代码 Demo Showcase

该项目用于展示两类典型低代码产品形态：

- 前端低代码：页面搭建、表单配置、组件拖拽、实时预览
- 后端低代码：数据模型设计、流程编排、动作链配置、接口编排

## 展示内容

- 前端页面画布与组件库
- 页面结构树与属性面板
- 后端模型设计器
- 流程引擎配置视图
- 动作链与接口编排视图
- Schema/配置结果展示
