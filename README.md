# 低代码 Showcase

这是一个拿来演示的低代码项目。

仓库里放了两套界面：

- 前端低代码：看页面怎么搭、表单怎么配、组件怎么拖
- 后端低代码：看模型怎么配、流程怎么走、动作和接口怎么编排

它的目标很简单，不是做成完整平台，而是让人一眼看明白：

- 前端低代码大概长什么样
- 后端低代码大概长什么样
- 两边各自负责什么

## 现在能看到的东西

- 前端模式里有组件库、页面画布、结构树和属性面板
- 后端模式里有模型设计、流程编排、动作链和接口编排
- 右侧面板可以直接改当前选中的配置
- 页面里保留了结果预览，方便讲解“配完之后会产出什么”

## 适合怎么用

- 给团队做方案演示
- 录一个产品概念视频
- 跟别人解释前端低代码和后端低代码的区别
- 当成继续往下做的平台原型
