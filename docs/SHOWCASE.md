# Demo 说明

## 原理

这个项目只展示两件事：

1. 前端低代码怎么搭页面  
前端侧通过组件库、画布和属性面板，把页面结构、表单字段和展示内容配置出来。

2. 后端低代码怎么配逻辑  
后端侧通过模型、流程、动作和接口编排，把业务处理逻辑配置出来。

前端低代码最后产出的是页面配置。  
后端低代码最后产出的是模型定义、流程定义、动作链或接口处理配置。

## 怎么用

1. 切到 `前端低代码`
2. 从左侧组件库拖组件到中间画布
3. 在右侧修改页面名称、路由和组件属性
4. 通过中间画布看页面效果

1. 切到 `后端低代码`
2. 在左侧选择要看的能力：
   `数据模型`、`流程编排`、`动作链`、`接口编排`
3. 在右侧修改当前资源配置
4. 通过中间区域和结果预览看配置产出
