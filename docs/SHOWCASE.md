# 低代码 Demo 说明

这个项目用于对外展示两类典型低代码形态：

- 前端低代码：页面、表单、组件、布局、实时预览
- 后端低代码：数据模型、流程编排、动作链、接口编排、连接器

## 展示重点

### 前端低代码

- 组件库拖拽到画布
- 页面结构树
- 右侧属性配置
- 页面 Schema 输出

适合说明：

- 活动页搭建
- 报名页配置
- 运营后台页面快速组装

### 后端低代码

- 数据模型设计
- 审批/任务流程编排
- 动作链配置
- 接口处理步骤编排
- 连接器接入

适合说明：

- 审批流系统
- 工单流转系统
- 数据同步任务
- 外部系统集成

## 实际落地方式

这类设计器通常有两种落地路径：

1. 运行时解释执行

- 将页面或流程配置保存为 Schema
- 运行时引擎读取 Schema
- 动态渲染页面、执行流程或编排动作

2. 代码/配置生成

- 将页面配置生成 React/Vue 页面
- 将模型配置生成表结构或 ORM 定义
- 将流程配置生成流程定义 JSON / DSL
- 将接口编排生成服务调用链

## 项目说明

这个仓库主要用来做演示。

如果你想快速给别人说明“前端低代码”和“后端低代码”分别是什么、界面通常怎么长、各自解决哪类问题，这个项目就是为这个目的准备的。

它现在更偏展示，不是完整的生产平台。运行时、权限体系、发布链路这些能力都没有展开做；这里主要保留了设计器最核心的外观和几种典型交互，方便拿来讲解、录屏或者做方案展示。
